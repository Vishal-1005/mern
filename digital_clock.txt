src/App.js
import React, { useState, useEffect } from "react";
import "./App.css";

function App() {
  const [time, setTime] = useState(new Date().toLocaleTimeString());

  // Update the time every second
  useEffect(() => {
    const timer = setInterval(() => {
      setTime(new Date().toLocaleTimeString());
    }, 1000);

    // Cleanup timer when component unmounts
    return () => clearInterval(timer);
  }, []);

  return (
    <div className="clock-container">
      <h1>Digital Clock</h1>
      <div className="clock">
        {time}
      </div>
    </div>
  );
}

export default App;

Here are short and clear answers:

1. What is useEffect used for in React?
useEffect is used to run side effects in a component (like timers, API calls, or updating the DOM).

2. Why do we use clearInterval inside useEffect?
To stop the interval when the component unmounts, preventing memory leaks and unwanted repeated execution.

3. What does useState do in this project?
useState stores and updates the componentâ€™s changing data (state), like the current time.

4. How often does setInterval run?
It runs repeatedly after the specified time delay (e.g., every 1000 ms = 1 second).

5. What function gives the local time as a string?
The toLocaleTimeString() function returns the current time as a local time string.
